A Streamlit web application that analyzes emotions in text using multiple machine learning models.

The system classifies user input into one of six emotions and displays the result with a confidence score.

---

## Features

- Supports multiple models:
  - LSTM
  - Naive Bayes
  - SVM
  - KNN
  - RNN
  - Random Forest
- Text input with real-time analysis
- Emotion classification with confidence score
- Example texts for quick testing
- Model selection from the sidebar
## Emotions Supported

| Code | Emotion  |
|------|---------|
| 0    | SADNESS |
| 1    | JOY     |
| 2    | LOVE    |
| 3    | ANGER   |
| 4    | FEAR    |
| 5    | SURPRISE|
